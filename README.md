# Student Placement Analysis Dashboard

## Dataset Selected
College Placement Records Dataset (Placement_Data_Full_Class.csv)

## Why I Selected This Dataset
I selected this dataset because it provides information about students' academic performance, work experience, placement status, and salary. It is useful for analyzing the factors that influence campus placements and employment outcomes.

## Three Business Questions Answered
1. Which degree type receives the highest average salary?
2. Which specialization has the highest number of placed students?
3. How does placement vary based on gender and degree type?

## Tools Used
- Python (Pandas)
- SQLite
- Power BI
- GitHub

## Key Insight
Science & Technology (Sci&Tech) graduates receive the highest average salary, while Marketing & Finance (Mkt&Fin) has the highest number of placed students.
